## Bottleneck Identification

```
BOTTLENECKS
============

CRITICAL: Processor Starvation
  - Mourning Chapel capacity: 600 grief/run
  - Grief generation: 120/run (per wound)
  - Deficit: -480 grief (processors idle 80% of time)
  - Impact: Wisdom generation rate-limited by input
  - Fix: Increase wound generation OR add more negative sources

CRITICAL: Calm Inflation
  - Generation: 350+/run
  - Consumption: 50/run
  - Excess: 300+/run
  - Impact: Calm becomes worthless, no tension
  - Fix: Add calm sinks OR make calm conditional requirements

WARNING: Energy Irrelevance After Day 10
  - Energy becomes irrelevant once buildings placed
  - No ongoing energy costs for processing
  - Impact: Energy system loses meaning
  - Fix: Add upkeep costs OR energy-gated abilities

WARNING: Insight Hyperinflation
  - Generated: 150+/run
  - Building unlocks: 25 max
  - Excess: 125+
  - Impact: Insight worthless after unlocks
  - Fix: Add insight sinks (research, buffs, abilities)

WARNING: Negative Emotion Scarcity
  - Events spawn bursts, but generators don't keep up
  - Processing capacity exceeds generation by 3-4x
  - Impact: "Mental health" feels too easy
  - Fix: Tune generator rates up OR add more negative sources
```

## Recommendations

```
ECONOMY BALANCE RECOMMENDATIONS
================================

1. [CRITICAL] Increase Negative Generation Rates
   Currently: Wound 0.12/s, Worry Loop 0.18/s
   
   Option A: Double generator rates
     - Wound: 0.24/s (1 grief every 4s)
     - Worry Loop: 0.36/s (1 anxiety every 2.8s)
   
   Option B: Add more negative generators per run
     - Events spawn persistent generators, not just bursts
     - Inciting incidents create 2-3 permanent wounds
   
   Option C: Reduce processor capacity
     - Halve processor speeds
     - Mourning Chapel: 8s instead of 4s

2. [CRITICAL] Add Calm Sinks
   Current sinks: Only Grounding Station (-1/cycle)
   
   Option A: Make processors consume calm
     - Mourning Chapel requires Calm x 1 to operate
     - Anxiety Diffuser requires Calm x 1 as catalyst
   
   Option B: Add calm decay
     - Calm decays at 0.05/second if total > 10
   
   Option C: Add calm-hungry buildings
     - "Serenity Amplifier" consumes Calm x 5/day for bonus effects

3. [IMPORTANT] Add Energy Upkeep
   Problem: Energy irrelevant after building phase
   
   Option A: Per-day processor costs
     - Processors cost 1 energy/day to operate
   
   Option B: Processing energy cost
     - Each processing cycle costs 0.5 energy
   
   Option C: Habit activation costs
     - Morning Routine costs 2 energy to activate

4. [IMPORTANT] Add Insight Sinks
   Problem: Insight hyperinflates after unlocks
   
   Option A: Research system
     - Spend 10 insight to unlock efficiency upgrades
   
   Option B: Building evolution
     - Spend insight to evolve buildings (Awakened state)
   
   Option C: Crisis mitigation
     - Spend insight to reduce event severity

5. [IMPORTANT] Rebalance Wellbeing Formula
   Problem: Too easy to reach Flourishing (80+)
   
   Option A: Increase normalizer
     - Change from 50 to 100
   
   Option B: Reduce derived weight
     - Change from 3.0 to 2.0
   
   Option C: Add decay to wellbeing
     - Wellbeing trends toward 50 over time without active management

6. [POLISH] Add Negative Feedback Loops
   Problem: Game gets easier as you progress
   
   Option A: Scaling generation
     - Negative generators speed up as wellbeing increases
   
   Option B: Complacency mechanic
     - Long periods of high wellbeing spawn new negative sources
   
   Option C: Event escalation
     - Later events spawn more/stronger negative sources
```
